V2X-Based Communication for Autonomous Vehicles Using CARLA

This project implements a Vehicle-to-Everything (V2X) communication framework in the CARLA simulator to enable real-time data exchange between autonomous vehicles and road infrastructure. The system supports Vehicle-to-Vehicle (V2V) and Vehicle-to-Infrastructure (V2I) communication, allowing vehicles to share information such as location, speed, direction, traffic signals, and road conditions.

By facilitating cooperative awareness and intelligent decision-making, the framework helps improve collision avoidance, traffic efficiency, and overall road safety. Various urban driving scenarios were simulated in CARLA to evaluate the effectiveness of V2X communication in connected and autonomous vehicle environments.

Features
Vehicle-to-Vehicle (V2V) communication
Vehicle-to-Infrastructure (V2I) communication
Real-time vehicle state sharing
Traffic signal and road condition awareness
Collision warning and hazard notifications
Autonomous driving simulation using CARLA
Technologies Used
CARLA Simulator
Python
V2X Communication
Autonomous Vehicle Simulation
Traffic Scenario Analysis

This project demonstrates how connected vehicle technologies can enhance autonomous driving performance and support the development of safer and more efficient intelligent transportation systems.
